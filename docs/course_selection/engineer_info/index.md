# 工信篇


!!! tip "工程图学基础"

    !!! abstract "朱继宏老师"
        *   **课程介绍：** 老师非常负责，会亲自指导学生画图。课程内容比较繁琐，需要耐心和细心。
        *   **课程任务：** 平时作业是画图，工作量较大，期末考试。
        *   **评价：** 想要学好画图的同学推荐选择。

    !!! abstract "田凌老师"
        *   **课程介绍：** 老师讲课细致，对学生要求严格。课程会介绍CAD软件的使用。
        *   **课程任务：** 作业量大，需要花费大量时间。
        *   **评价：** 课程较卷，但能学到很多实用的技能。
